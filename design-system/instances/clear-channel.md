# What
The canonical release of the core — the dark, monospace, single-accent
register the whole system was first written down to describe. Clear Channel
is computational minimalism with nothing overridden: void grounds, white
geometry, one spectral accent at a time, JetBrains Mono throughout. If a
later instance forks, this is the thing it forks *from*.

# Overrides
None. This instance *is* the core defaults made concrete.

| Axis | Value |
|------|-------|
| Ground | `--ground-void` → `--ground-modal` (the full dark elevation stack) |
| Accent | one spectral single per project (`--accent`) |
| Type | JetBrains Mono / IBM Plex Mono |
| Motion | physics-first; `--ease-out` only for interaction feedback |

# Provenance
[69-clear-channel](../../69-clear-channel/) — the visual-language reference
