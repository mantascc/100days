# What
A warm, muted release — the same structural core wearing a desaturated
purple-and-terra coat. Where Clear Channel runs cold and electric, Ube
Kaizen runs soft and earthen: violet primary, lavender secondary, terra
cotta tertiary. It tests the system's claim that the philosophy survives a
complete palette swap. It does.

# Overrides
Departs from Clear Channel chiefly in palette, and relaxes the strict
single-accent rule toward a small harmonized triad.

| Axis | Override |
|------|----------|
| Accent | `ube-violet #7b6fa5` (primary) |
| Secondary | `ube-lavender #9d91b0` |
| Tertiary | `ube-terra #9e6240` |
| Accent rule | three harmonized named colors instead of one spectral single |
| Type / motion / spacing | inherited from core, unchanged |
