Base unit: 4px (multiply). `active` tokens are the load-bearing rhythm of
most layouts; `provisional` ones appear rarely and may not survive.

| Token | Value | Usage | Status |
|-------|-------|-------|--------|
| `--space-1` | 4px | Micro gaps | active |
| `--space-2` | 8px | Tight padding | active |
| `--space-3` | 12px | Element gaps | active |
| `--space-4` | 16px | Standard margin | active |
| `--space-5` | 20px | Section padding | provisional |
| `--space-6` | 24px | Component gaps | active |
| `--space-8` | 32px | Large margins | active |
| `--space-10` | 40px | Canvas edges | active |
| `--space-12` | 48px | Major sections | provisional |
| `--space-16` | 64px | Generous breathing | provisional |
