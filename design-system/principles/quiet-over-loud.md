# Stand
An interface should recede so the system can be seen. No element competes
for attention it has not earned; the work asks to be watched, not noticed.
Quiet is the default register — flat grounds, single accent, no theatrics —
and everything loud must justify the cost of the noise it makes.

# Counterforce
Restraint can starve. A system so quiet it never declares itself becomes
illegible — the user cannot find the edge, the control, the state change.
Presence is a legitimate force: a moment must sometimes shout so the rest
can stay quiet. The pull here is between a calm whole and a readable part.

# When to break
Break for a single, load-bearing signal: an error that must be obeyed, a
focus ring that must be found, one accent doing the work of orientation.
Break when quiet has tipped into ambiguity and the user is lost. Never
break for decoration — only to restore legibility the quiet took away.
