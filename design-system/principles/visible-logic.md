# Stand
Show the rule, not just the result. The pleasure is in watching something
think — local rules made visible, complexity arriving as a consequence the
viewer can trace back. The mechanism is the subject; the surface is a
window onto it. A sketch that hides its logic is a magic trick, not a system.

# Counterforce
Total transparency is noise. Expose every variable and the rule drowns in
its own instrumentation; the emergent result — the thing actually worth
seeing — gets buried under the scaffolding that produced it. Richness
sometimes needs the machinery to step back so the pattern can step forward.

# When to break
Break when the mechanism is trivial and the result is the point — let the
emergence speak and hide the plumbing. Break when showing the logic would
require chrome that violates quiet-over-loud. The rule stays implicit when
it is self-evident from one second of watching.
