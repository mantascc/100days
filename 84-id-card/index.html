Pointer input
normalization

Normalizes pointer inputs (position, pressure, tilt) to the [0,1] interval and applies the response curve declared on the target surface. Per-surface tolerance windows are read from §2.3. Output is consumed by the compositor and does not propagate beyond the surface boundary.
