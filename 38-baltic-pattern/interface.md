# Day 38: Generative Baltic Pattern

## Idea
Generative pattern generator creating Baltic folk art inspired geometric designs

## Description
This project generates patterns inspired by traditional Baltic folk textiles and ornaments. Using Inter font and clean interface with burgundy accents, the generator creates geometric patterns based on Baltic design principles. The aesthetic features cream background (#faf9f6), burgundy accent (#8b2d2d), and rounded modern UI elements. Info button provides pattern documentation.

## Data Concepts
- **Primary**: Visual (geometric patterns, folk art, generative design)
- **Secondary**: Spatial (pattern tiling, symmetry)
