# 77 · ASCII Sheet Documentation

A promotional documentation sketch for the ASCII Sheet component: a fixed top banner that filters the page underneath and renders an animated ASCII texture above it.

The page demonstrates the component as a product surface rather than a raw test: editorial masthead, operational dashboard ribbon, music/event signal, portfolio index, preset gallery, and a compact configuration snapshot.

## Files

- [index.html](index.html) — self-contained promotional demo and playground
- [interface.md](interface.md) — component anatomy, states, presets, and implementation notes

## Run

Open `index.html` directly in a browser, or serve the repository root with a static file server.

```sh
python3 -m http.server 3333
```

Then open http://localhost:3333/77-ascii-sheet-documentation/.
