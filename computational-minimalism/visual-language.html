chapter contract
every variation unit exposes the same interface. the loader never knows what a chapter draws — it only calls setup, builds the dom, and runs the loop. 
// chapter interface
{
  labels:  string[]                           // what each variant is called
  drawFns: Array<(cv, t) => void>             // one draw function per cell
  dt:      number                             // 0 = static; >0 = animated
  setup:   (id: string) => {canvas, ctx, W, H} // canvas initializer
}

01
glyph ramps
character density as continuous tone. a value [0,1] maps to a glyph index. the ramp is the palette — each vocabulary carries a different texture.
live mapping
○
0.50
02
value → property
six primitive encodings. each maps a normalized value to a single visual channel. one value, one property — never overload a channel.
03
threshold vs continuous
two mapping modes. continuous: value flows through. threshold: value snaps to on/off at a cut point. piecewise: multiple thresholds create bands. same data, three reads.
continuous
threshold
piecewise
04
signal binding
live inputs (audio fft, camera, mouse, sensors) wire to visual parameters through named bindings. multiplexing: one signal drives multiple outputs. demultiplexing: one output is driven by band-split inputs.
bass → speed
mids → density
treble → brightness
amp → alpha
live demo — move mouse to simulate signal
05
field rendering
spatial functions as data sources. sin, cos, atan2, noise — evaluated per cell, rendered through a glyph ramp. the function is the dataset. the grid is the viewport.
06
time encoding
temporal data becomes spatial through three patterns: grid (time × category), trail (position + decay), accumulation (value builds over frames).
grid — day × agent
trail — position + decay
accumulation — value over frames
07
relation encoding
connections between entities. force-directed layout lets topology become geometry. node = entity, edge = relation, distance = similarity. the physics finds the form.

const field = new CCField(canvas, {
  mode: 'think',        // think | breathe | rain | scan | ripple | progress
  ramp: 'circles',      // circles | blocks | braille | wave | crystal
  accent: '#00ffaa',
  fontSize: 11,
  speed: 1,
  density: 1
});
field.start();
field.setProgress(0.7); // progress mode
field.setMode('rain');  // switch live
