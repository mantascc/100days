cc-field

canvas ascii field — loading / thinking / progress component
modes
think — interference waves
breathe — radial pulse
rain — falling particles
scan — horizontal sweep
ripple — mouse-reactive
progress — fill with data
sizes
inline (120×16)
badge (40×40)
bar (280×20)
card (200×80)
glyph ramps
circles
blocks
braille
playground
usage
const field = new CCField(canvas, { mode: 'think', // think | breathe | rain | scan | ripple | progress ramp: 'circles', // circles | blocks | braille | wave | crystal accent: '#00ffaa', // any hex color fontSize: 11, // glyph size in px speed: 1, // animation speed multiplier density: 1, // spatial frequency multiplier opacity: 0.8 // global alpha }); field.start(); // begin animation loop field.stop(); // pause field.setProgress(0.7); // for progress mode [0,1] field.setMode('rain'); // switch mode live field.destroy(); // cleanup
