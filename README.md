# Hello world

100 day routine to explore computational aesthetics, data expression, coding concepts. Inspired by [Tiantian Xu](https://www.instagram.com/tiantianxu/)

## Structure

- `NN-name/` — numbered days, each self-contained (`index.html` + `interface.md`)
- `daily-sketch/` — incubator for unnumbered sketches; they graduate to numbered days
- `index.html` — the canonical gallery (alternative landing designs live in `landing-variants/`)
- `projects.json` — registry of graduated days
- `corpus.json` — machine-readable index compiled from the layers above; rebuild with `node build-corpus.js`
- `aesthetic-genome/`, `interface-query/`, `computational-minimalism/` — system layers over the corpus

Day 48 was never made — the gap is intentional history. See [`ROADMAP.md`](ROADMAP.md) for where this is headed.
