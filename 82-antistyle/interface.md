# Day 82: Antistyle Gallery

Museum gallery experience for antistyle as anti-illusion.

The interaction is a horizontal gallery walk. The first active room makes the
core paradox tangible: the viewer tries to remove style signals, and every
removal is immediately recorded as a new style decision. The later rooms then
represent a lineage of refusals: Dada against aura, radical design against good
taste, punk against polish, web brutalism against the rounded consensus, the
absorption trap, visible machinery, and the rough mark as evidence of human
decision.

Interface notes:

- Keep browser mechanisms visible: scrollbars, default links, telemetry, outlines.
- Show machinery by default. No reveal toggle; the exposed layer is the normal
  state.
- Let the gallery be partially tasteful, then interrupt that taste with exposed
  construction.
- Add entropy as live annotation drift: crooked labels, visible grids, source
  rails, and generated wall tags.
- Horizontal gallery movement supports drag-to-scroll. The default cursor is
  grab, dragging switches to grabbing, and interactive regions can opt out with
  `data-drag-scroll-ignore` plus a `data-cursor` value.
- A floating `interface.md` button opens these notes in a dialog, making the
  design brief part of the gallery surface.
- Make the paradox interactive: removing the frame, palette, typography, or
  explanation must generate a new label or rule.
- Use the plaque voice as an unreliable institution.
- Include one live surface where the viewer can make a crude mark.
- The point is not ugliness. The point is refusing the smooth fiction.
