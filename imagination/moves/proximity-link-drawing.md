# What
Draw an edge between two agents whenever they fall under a distance threshold, with the line's alpha faded by separation so links bloom and dissolve as the field moves. It's the single gesture that turns a particle cloud into a living mesh — the signature "constellation" look. Cheap, expressive, and so recognisable it's nearly a fingerprint of the practice's network sketches.

# Charge
High but overfamiliar — it makes anything look "generative" instantly, which is precisely the trap. A strong candidate for the Break strategy.

# Prompts
- Links that *persist* once formed — a memory mesh that accretes history.
- Threshold as the variable: a field where the connection distance breathes.
- Replace the line with something stranger — a gradient, a force, a sound.
