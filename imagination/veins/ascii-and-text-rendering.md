# What
Continuous fields rendered as ordered glyphs — sine and interference patterns mapped across a density ramp into a grid of characters. What began as a single ASCII sketch hardened into a reusable primitive: the "ASCII sheet" now serves as both an aesthetic and a documentation surface. The constraint (a fixed character set, a monospace grid) is the appeal; the maker likes how much expression survives such a coarse, legible quantiser.
