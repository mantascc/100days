# What
How do you make sound drive a visual so the motion feels *inevitable* rather than decorative — not "bars that wiggle to music"? The partial answers the practice has found: renormalise velocity so the flow stays controlled instead of exploding on a loud transient, smooth the signal with an EMA so it reads as motion not jitter, and aim for movement that feels like "a consequence of the optics, not decoration." The frontier is the moog-phase direction: sound as a phase portrait, structure rather than reaction.

# Charge
Quietly persistent — solid technique, unresolved aesthetics. Each new audio sketch presses on it again, which is exactly its job.

# Prompts
- Sound that changes a *rule*, not a value — the law of the system listens.
- A visual where you can't tell which feature the audio drives — fully fused.
- Map silence: build the piece around the signal's gaps, not its peaks.
