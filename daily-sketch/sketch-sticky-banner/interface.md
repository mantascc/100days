# sketch-sticky-banner

## idea
Variant of the 100days index where the Hello world banner is sticky and elevated one level above the project list — content slides under it on scroll.

## tags
layout, scroll, elevation, index-variant

## stack
vanilla · IBM Plex Mono · canvas/CSS

## motion
ASCII field banner continues to animate while pinned. Project list scrolls under a slightly opaque/elevated banner surface.
