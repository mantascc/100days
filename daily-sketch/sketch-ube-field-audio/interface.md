# sketch-ube-field-audio

## idea
Single ube-purple ASCII wave field, full viewport, driven by microphone input — bass controls speed, mids control density, amplitude controls brightness

## tags
audio-reactive, ascii, generative, animation, microphone

## stack
vanilla · IBM Plex Mono · canvas · Web Audio API

## motion
Continuous sine wave interference field; wave speed, frequency, and glyph brightness respond to live mic input in real time
