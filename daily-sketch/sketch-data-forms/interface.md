# sketch-data-forms

## idea
Same dataset, four chapters — each chapter renders it eight ways. Fixed data, varied representation.

## tags
data-visualization, variation-grid, chart-types, encoding, generative, interactive

## stack
vanilla · JetBrains Mono · canvas/CSS · chapter/factory pattern

## motion
Animated where the representation benefits from it (force-directed layouts, live encoding transitions). Static where clarity matters more (charts, projections).

## chapters
1. **chart-types** — line, bar, scatter, area, heatmap, step, dot-plot, radial
2. **graph-layouts** — same node/edge set, 8 force-directed physics configurations
3. **projections** — same geographic data, 8 map projections
4. **encodings** — same column of values, 8 visual encodings (position, color, size, angle, density, order, shape, texture)
