A reusable elevated surface. Animated ASCII texture stacked over a backdrop filter, with an optional punch-through label so type stays legible.

Drop it in as a header, a modal backdrop, a toast, a divider, a side drawer, a marquee, an image-reveal curtain, a page-transition wipe. One component, many shapes — keep scrolling.

Anatomy

Four layers, stacked.

A transparent strip captures the position. A backdrop filter processes what passes underneath. A canvas paints animated glyphs on top. An opaque label punches through both for legibility.

Hello world
  • 01 .sheet  — transparent strip, captures pointer events position: fixed/absolute · z-index: 10
  • 02 .filter-sheet  — processes what's underneath backdrop-filter: blur(10px) | url(#distort)
  • 03 canvas  — animated glyph field destination-out trail · pluggable formula · 20fps
  • 04 .sheet-label  — opaque punch-through proximity field shrinks glyphs near edges

Formula gallery

Swap the math, change the mood.

The formula maps grid position and time to a value 0..1; the glyph ramp samples that value. Five presets ship by default — swap in your own function for custom motion.

interference · default (x, y, t) => (sin(x·0.1 + t) + cos(y·0.12 − t·0.7) + sin((x+y)·0.05 + t·0.5) + 3) / 6
ripple (x, y, t, cx, cy) => (sin(distance(x,y → center) · 0.3 − t · 2) + 1) / 2
noise · pseudo (x, y, t) => (sin(0.23x + 0.17y + t) + sin(0.07x − 0.13y − 0.5t) + sin(0.31(x+y) + 1.3t) + 3) / 6
flow (x, y, t) => (sin(0.15x + 2·sin(0.1y) + t) + cos(0.08y + 0.5t) + 2) / 4
spiral (x, y, t, cx, cy) => (sin(r · 0.3 + θ · 3 − t · 2) + 1) / 2

Embed

Three surfaces to tweak.

Drop in with HTML attributes, override with JS config, or open the built-in tweak panel for live prototyping. 

<ascii-sheet height="200" color="#7b6fa5" filter="blur" formula="interference">
  <span slot="label">Hello world</span>
</ascii-sheet>
sheet.config = {
  texture:   { glyphs: [' ', '·', '∙', '∘', '○', '◌', '◎', '⊙', '●'], fontSize: 11, fps: 20 },
  animation: { formula: (x, y, t) => myField(x, y, t), color: '#7b6fa5', speed: 0.025 },
  filter:    { mode: 'blur', blurPx: 10, tintAlpha: 0.4 }
};
