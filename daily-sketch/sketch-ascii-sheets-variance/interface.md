# sketch-ascii-sheets-variance

## idea
Promotional showcase site for the ASCII sheet component — one primitive, many shapes (headers, modals, toasts, dividers, drawers, image-reveals, marquees, page transitions).

## tags
component-showcase, ascii, typography, layout, scroll, generative, landing-page

## stack
vanilla · IBM Plex Mono · canvas/CSS

## motion
Each section runs its own ASCII sheet with a different formula / filter / orientation. Scroll progress modulates the hero's wave field; mouse interaction reveals the punch-through label. Multiple instances on the page animate independently.
